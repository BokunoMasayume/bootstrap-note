    模态弹出窗主体内容

  • 不同尺寸，添加.modal-lg .modal-sm类添加到.modal-dialog容器上。
  • 自定义属性触发模态弹出框的两种方法。
    • 声明式触发 ：给触发元素(一般button，实测span也行)添加两个属性data-toggle="modal" data-target=".modal的类或#id"
    • <a>触发。和声明式触发的区别就是用href属性代替了data-target属性，连属性值都不用变orz。
  • 为弹出框增加过渡动画效果.fade。<div class="moal fade" tabindex="-1">

    • Tips:
    tabindex属性可以设置键盘中TAB键在控件中的移动顺序，即焦点顺序。
    当值为1~32767中的任一个，焦点将由较小属性值移动到较大属性值，属性值一样时，按代码中出现顺序为准。
    当值为0（默认值）时，排到上述正值之后。
    当值为-1时，它将被排除在序列之外，也就是会被TAB键忽略。

  • 模态弹出框常用data-参数说明一般使在触发元素上。
    属性名称 类型 默认值 描述 备注
    data-toggle 字符 modal 用来控制模态弹出框的显示。 必须要在触发modal的元素上定义，并且值必须是modal。
    data-target 字符 用户自定义 用来定义被触发的模态框是哪个，值可以是.类名或#id。 必须要在触发modal的元素上定义，且能与其值对应的对象必须只有一个
    data-backdrop 布尔值 true 是否包含一个背景div元素。true时单击该背景关闭弹出窗；static时单击背景不会关闭弹出窗。
    data-keyboard 布尔值 true true时按键盘的ESC键可以关闭弹出窗；false时无法用ESC键关闭弹出窗。
    data-show 布尔值 true 初始化时，弹出框是否显示。
    href URL路径 空值 如果用a元素触发弹出框，且其href值不是以#开头，就说明这是个url地址，模态弹出框会用加载的其内容替换原有的modal-content的内容，这时data-target的值必须是id
  • JS触发模态弹出框。给按钮或随便什么你看上的元素绑个click或随便什么事件，函数里调用.modal元素的modal()方法就行了。栗子： 
            $(function(){
          $(".btn").click(function(){
            $("#mymodal").modal();
          });
        });
  • JS触发时的属性设置。
    属性 对应的data-属性
    backdrop data-backdrop
    keyboard data-keyboard
    show data-show
    remote href 
            $(function(){
          $(".btn").click(function(){
              $("#mymodal").modal({
                keyboard:false
              });
          });
        });
  • JS触发时参数设置。
    参数 使用方法 描述
    toggle 
                  $("#themodal").modal("toggle");
    		触发时，反转弹出框状态，原本显示则关闭；原本关闭则显示。
    show 同上 触发时显示弹出框
    hide 同上 触发时关闭弹出框
  • JS触发时事件设置。模态弹出窗支持四种类型的事件，分别是模态弹出窗的弹出前、弹出后，关闭前、关闭后。可以弹出后出个动画之类的hhh
    事件类型 使用方法 描述
    show.bs.modal 
                  $('#themodal').on("show.bs.modal",function(){
                //高深莫测的代码...
            });//除了on方法还可以用bind方法。
    		在show方法调用时立刻触发；如果单击了一个元素，将该元素作为事件的relatedTarget属性。
    shown.bs.modal 同上 在模态弹出框完全显示（css动画也加载完成）后触发；如果单击了一个元素，将该元素作为事件的relatedTarget属性。
    hide.bs.modal 同上 在hide方法调用（但还没来得及关闭隐藏）时立即触发
    hidden.bs.modal 同上 在模态弹出框完全隐藏后触发。

轮播图

  • 轮播图引入地址：<script src="http://cdn.bootcss.com/bootstrap/2.3.1/js/bootstrap-carousel.min.js"></script>
  • 一个轮播图主要包含三个部分：
    • 轮播的图片。
    • 轮播图片计数器。
    • 轮播图片的控制器。
  • 轮播图框架的使用方式。
    1. 设计一个轮播图的容器(div)，给它添加.carousel类。
    2. 设计轮播图的计数器，在上条创建的容器里添加一个.carousel-indicators类的ol列表。里面每个li对应轮播图的一个图片。
    3. 设计轮播图的播放区。在第一条的容器(div)里添加一个.carousel-inner类的div,里面每个.item类容器放置一个图片。
    4. 在item容器里除了图片链接，还可以添加标题和描述内容。.carousel-caption
    5. 设计控制器。.right|left .carousel-control 
                  <a class="right carousel-control" >
              <span class="glyphicon-chevron-right"></span>
            </a>
            <a class="left carousel-control" >
              <span class="glyphicon-chevron-left"></span>
            </a>
  • 声明式触发。
    属性 取值 描述
    data-ride carousel 定义在.carousel容器上。
    data-target carousel容器的类或id。 定义在计数器每个<li>上。
    data-slide prev或next 定义在控制器的<a>上，同时<a>上的href属性取值carousel容器的类或ID.
    data-slide-to 从0开始的整数。 定义在计数器的每个li上
    data-interval number  默认5000 幻灯片轮换的等待时间(毫秒)，若为false，则不会自动循环。
    data-pause string   默认值hover. 默认鼠标悬停时停止播放，离开开始播放。
    data-wrap boolean  true 轮播是否持续循环。
