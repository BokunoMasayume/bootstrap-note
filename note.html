表单

    表单的每个输入区域最好放在一个div中，且带一个from-group类
  • 水平表单。即标签在左，控件在右。利用form标签上的.form-horizontal类实现，需配合bootstrap的网格系统。
  • 内联表单。即表单的所有控件在同一行显示 .form-inline类实现。
  • 表单控件：
    • 给各种控件加.form-control 类：
    • bootstrap中的checkbox和radio有点特殊，在使用它们的时候在<label>标签外包一层<div class="radio|checkbox">可增加页面颜值。
    • 使复选框，单选按钮水平排列：给label加类.checkbox-inline .radio-inline即可。外面不要包.ckeckbox|radio的div了。
    • 文本域<textera>。rows属性设置高度，cols属性设置宽度，若在<textera>中添加.form-control类则宽度为100%或auto。
    • 按钮。bootstrap框架中的按钮一般采用<button>实现的。除此之外，给a,input，甚至span，div等其他标签添加bootstrap的按钮相关类也可以制作标签。
      style
      class btn default btn btn-primary btn btn-info btn btn-success btn btn-warning btn btn-danger btn btn-inverse btn btn-link
      可以通过给按钮添加btn-lg,btn-sm,btn-xs类快捷的改变按钮大小。
      块状按钮。给按钮添加.btn-block类实现。 按钮状态。分为活动状态和禁用状态。禁用状态通过给<button>添加disabled属性或者添加.disabled类就能做到。活动状态分为悬浮状态:hover,点击状态:active和焦点状态:focus，是通过伪类实现的。
    • 表单控件大小。可以通过设置控件的height,line-height,padding,font-size等属性设置控件的高度。bootstrap框架提供了两个类可以使控件更大.input-sm或更小.input-lg>。这两个类适用于input,textarea,select控件。
    • 焦点状态，通过加:focus伪类定义样式。禁用状态，表单禁用通过给控件加disabled属性实现。bootstrap对这些的样式有一些优化。
      form标签中<fieldset disabled>标签括起来的区域会全部禁用，但如果其内部有legend标签并且legend标签里有输入框，这个输入框无法被禁用。
    • 验证状态。bootstrap提供了三种样式.has-warning .has-error .has-success，使用时只需要在form-group容器上添加对应类名。
    • 表单提示信息。在.form-group容器里加入一个.help-block样式的信息容器。 
          <div class="form-group has-success ">
	<label class="control-label" >姓名：
		<input type="text"  class="form-control" placeholder="输入姓名"/>
	</label>
	<span class="help-block">输入格式正确</span>
    </div>

导航条

    导航(nav)和导航条(navbar)名字很像，但在bootstrap框架还是有明显的区别。
    比如导航条有背景色，而且导航条既可以是纯链接(就像导航)，也可以是表单，甚至两种的混合等。
  • 基础导航条。基础导航条以导航为基础。把基础无样式导航变成导航条需两步
    1. 在导航的<ul|ol class="nav">标签增加.navbar-nav类
    2. 在导航的列表外部包一个div，并为其添加.navbar .navbar-default类<div class="navbar navbar-default">
  • 为导航条添加标题。
    1. 在列表（ul）前插入一个div盒子，并添加.navbar-header类<div class="navbar-header">
    2. 在上一步添加的div中放入一个a标签，并为其添加.navbar-brand类<a class="navbar-brand">标题</>
